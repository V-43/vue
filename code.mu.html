2.Основы работы с фреймворком Vue

Задача 1

Пусть в data хранится хранится имя и фамилия пользователя. Выведите данные пользователя на экран в каком-нибудь абзаце.

Ответ:

{{ name }} {{ surname }}

Задача 2

Пусть в data хранится число num=5. Выведите квадрат этого числа.

Ответ:

{{ num * num }}

Задача 3

Пусть в data хранятся числа num1=1, num2=2, num3=3. Выведите сумму этих чисел.

Ответ:

{{ num1 + num2 + num3 }}

Задача 4

Пусть в data хранится следующий массив: [1,2,3]. Выведите сумму элементов этого массива.

Ответ:

{{ arr.reduce((sum, current) => sum + current, 0) }}

Задача 5

Пусть в data хранится следующий объект: obj: {a: 1, b: 2, c: 3}. Выведите сумму элементов этого объекта.

Ответ:

{{ Object.values(obj).reduce((sum, current) => sum + current, 0) }}

Задача 1

Ответ:

  • {{ elem }}

Задачи 2-3

Дан массив имен ['user1', 'user2', 'user3']. Выведите каждый элемент этого массива в отдельной li в списке ul так, чтобы в каждой li кроме элемента массива еще и стоял его порядковый номер(+1) в массиве.

Ответ:

  • {{ elem }} - {{ key + 1 }}

Задачи 4-5-6-7

Дан следующий объект: { employee1: '100$', employee2: '200$', employee3: '300$', }. Выведите зарплаты работников в виде списка ul.

Ответ:

  • {{ employee }} - {{ salary }} - {{ idx + 1 }}

Задача 9

Дан массив с продуктами (названия, цены за штуку и количество). 

                products: [
                    {name: 'product1', price: 100, quantity: 5},
                    {name: 'product2', price: 200, quantity: 4},
                    {name: 'product3', price: 300, quantity: 3},
                ]
Выведите названия, цены, количества и сумму (произведение цены и количества).

Ответ:

  • {{ product.name }}, цена: {{ product.price }}, количество: {{ product.quantity }}, сумма: {{ product.quantity * product.price }}
